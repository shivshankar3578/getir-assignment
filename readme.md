# getir-assignment
App is deployed on heroku at https://getir.herokuapp.com/ 

## Developement
### setup env
```export mongo_uri=your_mongodb_connection_string```
### checkout 
```git clone https://github.com/shivshankar3578/getir-assignment```
### install dependancies

```
cd getir-assignment
npm install
```
### start local server
``` npm start```


### test
``` npm test```
